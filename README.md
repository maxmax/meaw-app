## Simple server implementations included

> All serve static files from `public/`

> Handle requests to `/api/comments` to fetch or add data.
Start a server with one of the following:

### Node
```sh
npm install
node server.js
```

#### Ruby
```sh
ruby server.rb
```

#### PHP
```sh
php server.php
```

### Changing the port

```sh
PORT=3001 node server.js
```
